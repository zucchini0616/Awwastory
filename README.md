## Local Setup
1. Clone the repo
  ```bash
git clone https://github.com/zucchini0616/Awwastory.git
cd Awwastory

```
2. Initialize & install
 ```bash
npm init -y
npm install
```
3. Fix audit issues (optional)
```bash
npm audit fix
```
4. Start the server
```bash
npm start
```
5. Verify
```bash
http://localhost:3000/index.html
```
